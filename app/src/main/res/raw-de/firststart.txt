Die App sucht momentan den openHAB Server im lokalen Netz. Sobald der Server gefunden wurde, wird die Webseite geladen. Bitte fahren Sie dann mit der initialen Konfiguration von HABPanel fort.
Benutzen sie das Kontextmenü, um die gewünschte Seite als Startseite zu markieren.

Wenn der Server nicht gefunden werden kann, öffnen Sie das Menü und geben Sie die URL in den Einstellungen ein.

Um das Menü zu öffnen, ziehen Sie den Finger vom rechten Rand zur Mitte des Bildschirms.
